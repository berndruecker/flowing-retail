## Description

A [Nest](https://github.com/nestjs/nest) implementation of the Flowing Retail demo, using the [NestJS Zeebe integration](https://www.npmjs.com/package/@payk/nestjs-zeebe).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
